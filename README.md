# 🇵🇰 Pakistan Travel Chatbot

An AI-powered chatbot that provides document-grounded travel guidance for Pakistan using OpenAI and LangChain.

## ✨ Features
- Answers based on a reference PDF
- Friendly, culturally respectful responses
- Session memory using LangChain
- UI in Colab using widgets
